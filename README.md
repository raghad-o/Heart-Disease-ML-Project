# Heart Disease Machine Learning Analysis

## Overview
This project is an individual machine learning study on a heart disease dataset from Kaggle.  
The goal is to explore, analyze, and compare different classification models to predict heart disease.

Dataset used:  
https://www.kaggle.com/datasets/johnsmith88/heart-disease-dataset

---
# Workflow

- Data loading and exploration  
- Data cleaning and preprocessing  
- Feature scaling  
- Train/test split (80/20)  
- Training models: Naive Bayes, Decision Tree, KNN (Euclidean & Manhattan)  
- Model evaluation  
- Cross-validation (5-fold)  
- ROC curve and AUC analysis  
- Decision boundary visualization  

---

## Evaluation Metrics
- Accuracy  
- Precision  
- Recall  
- F1-score  
- ROC-AUC
